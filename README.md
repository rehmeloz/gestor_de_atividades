# 📋 Gestor de Atividades

Um sistema de gerenciamento de atividades com operações **CRUD**, desenvolvido com uma arquitetura moderna separando frontend e backend.

## 🚀 Tecnologias Utilizadas

### Frontend
- **React** — Biblioteca para construção da interface de usuário
- **JavaScript (JS)** — Linguagem principal do frontend

### Backend
- **C#** — Linguagem principal da API
- **Entity Framework Core (EF Core)** — ORM para acesso e gerenciamento do banco de dados SqlLite

## ✨ Funcionalidades

- ✅ **Criar** novas atividades
- 📄 **Listar** todas as atividades cadastradas
- ✏️ **Editar** atividades existentes
- 🗑️ **Deletar** atividades
